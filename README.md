# ISCLNet

X. He, L. Fang, M. Tan, and C. Xiang, "Intra- and Inter-Slice Contrastive Learning for Point Supervised OCT Fluid Segmentation," IEEE Transactions on Image Processing, vol. 31, pp: 1870 - 1881, 2022.
